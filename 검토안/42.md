초기 세팅의 핵심은 “처음부터 정답 파라미터를 맞히는 것”이 아니다.

```text
넓게 탐색하되,
나쁜 구간은 강하게 차단하고,
성과가 확인된 조합만 좁혀가는 구조
```

# 1. 초기 전략 구성

최소 4개 엔진으로 시작하는 게 좋다.

| 전략                  | 목적          |
| ------------------- | ----------- |
| Pullback trend long | 상승 추세 주력    |
| Breakout long       | 강한 추세 확장 포착 |
| VWAP mean reversion | 횡보장 수익      |
| Futures short       | 하락장 대응      |

처음부터 너무 많은 전략을 넣으면 원인 분석이 어려워진다.
4개면 충분히 넓고, 추적 가능하다.

# 2. 레짐 기본값

초기 레짐은 단순하고 강하게 잡는 게 좋다.

| 레짐        | 조건 예시                               |
| --------- | ----------------------------------- |
| uptrend   | EMA50 > EMA200, slope 양수, ADX 중간 이상 |
| range     | ADX 낮음, 가격이 VWAP 주변, 변동성 안정         |
| downtrend | EMA50 < EMA200, slope 음수            |
| panic     | 단기 변동성 급등, spread 확대                |
| quiet     | 거래량/변동성 부족                          |

중요:

```text
panic / quiet / data bad 구간은 무조건 no-trade
```

# 3. 초기 파라미터 탐색 범위

초기에는 좁게 잡으면 edge를 못 찾는다.
다만 live에는 절대 직접 연결하지 않는다.

| 항목                          | 초기 탐색 범위                      |
| --------------------------- | ----------------------------- |
| ATR stop                    | 0.8 ~ 2.5 ATR                 |
| take profit                 | 1.0 ~ 4.0 ATR                 |
| trailing start              | 1.0 ~ 2.5 ATR                 |
| pullback depth              | 0.3 ~ 1.5 ATR                 |
| breakout threshold          | 최근 range high + 0.2 ~ 1.0 ATR |
| VWAP band                   | 1.0 ~ 2.5 sigma               |
| time stop                   | 3 ~ 12 candles                |
| ADX threshold               | 15 ~ 30                       |
| volume expansion            | 1.2x ~ 3.0x                   |
| signal confidence threshold | 0.55 ~ 0.80                   |

15분봉 기준이면 time stop 3~12 candles는 약 45분~3시간이다.

# 4. 종목 선정 초기값

처음에는 너무 많은 종목을 돌리기보다 후보군을 동적으로 좁혀야 한다.

초기 필터:

```text
거래대금 상위
스프레드 낮음
호가 잔량 충분
최근 변동성 있음
극단적 펌핑 직후 제외
스테이블코인 제외
```

랭킹 점수:

```text
symbol_score =
liquidity_score
+ volatility_score
+ relative_strength_score
+ volume_expansion_score
+ spread_quality_score
+ regime_fit_score
```

초기 감시 수:

```text
Upbit: 상위 20~40개
Binance spot: 상위 30~60개
Binance futures: 상위 30~80개
```

실제 진입 후보는 더 줄인다.

```text
entry candidates: 상위 3~10개
```

# 5. 신호 등급 초기값

처음에는 약한 신호를 과감히 버려야 한다.

```text
weak: 거래 금지
normal: 소액 paper only
strong: paper/shadow 핵심 후보
exceptional: sizing 후보
```

초기 live 후보는 strong 이상만 허용한다.

# 6. 기대값 필터

초기부터 반드시 비용 차감 후 판단해야 한다.

```text
net_expected_edge =
expected_move
- fee
- spread_cost
- slippage_cost
- funding_cost
```

초기 진입 조건:

```text
net_expected_edge > 0
```

월 10퍼센트 목표라면 더 엄격하게:

```text
net_expected_edge >= 1.5x estimated_total_cost
```

# 7. 손절/익절 기본 구조

각 포지션은 반드시 종료 조건을 가져야 한다.

초기 권장:

```text
stop_loss: 1.0 ~ 1.8 ATR
take_profit_1: 1.0 ~ 1.5 ATR
take_profit_2: 2.0 ~ 3.5 ATR
partial_take_profit: 30% ~ 50%
trailing_stop: TP1 이후 활성
time_stop: 6 ~ 12 candles
```

좋은 구조:

```text
TP1 도달 → 일부 익절
stop을 breakeven 근처로 이동
남은 물량은 trailing
```

# 8. 전략별 초기 방향

## Pullback trend

진입 조건:

```text
uptrend
EMA50 > EMA200
가격이 EMA20/50 또는 VWAP 근처로 눌림
재상승 확인
거래량 회복
```

금지:

```text
EMA50 아래 강한 이탈
ADX 급락
시장 전체 downtrend
```

## Breakout

진입 조건:

```text
range high 돌파
volume expansion
realized volatility expansion
close confirmation
spread 정상
```

금지:

```text
돌파 직후 과도한 장대봉
호가 얇음
slippage 예상 초과
```

## VWAP mean reversion

진입 조건:

```text
range regime
VWAP band 이탈
반전 캔들 또는 orderflow 회복
```

금지:

```text
trend regime
breakout 발생
volume expansion이 추세 방향으로 붙음
```

## Futures short

진입 조건:

```text
downtrend
breakdown
반등 실패
volume confirmation
funding/cost 문제 없음
```

금지:

```text
급락 직후 추격 short
panic spread
급반등 신호
```

# 9. 초기 risk 설정

초기에는 수익률보다 생존성과 데이터 수집이 우선이다.

```text
risk per trade: 0.1% ~ 0.3%
daily loss cap: 1.0%
weekly loss cap: 3.0%
monthly hard stop: 8.0%
max concurrent positions: 3~5
max exposure per symbol: 10%~20%
```

paper/shadow에서는 넓게 실험하되, live 후보 평가는 위 기준으로 해야 한다.

# 10. Shadow 후보 수

너무 많으면 분석이 흐려진다.

초기 권장:

```text
전략별 5~20개 parameter set
총 30~80개 shadow 후보
```

단, live 후보로 올라갈 수 있는 것은 상위 소수만.

```text
strategy별 top 1~3개만 refinement
```

# 11. 탈락 기준

초기부터 자동 탈락을 강하게 둬야 한다.

탈락 조건:

```text
net expectancy <= 0
profit factor < 1.1
trade count 충분한데 drawdown 과도
fee/gross profit 비율 과도
slippage 민감도 높음
특정 종목 1~2개에만 성과 집중
레짐 하나에서만 우연히 수익
```

# 12. 승격 기준

초기 live-ready 후보는 보수적으로 잡는다.

```text
trade count >= 100
net expectancy > 0
profit factor >= 1.25
max drawdown 허용 범위 내
fee/slippage 반영 후 양수
레짐별 손실 폭 제한
out-of-sample 또는 최근 구간 검증 통과
```

월 10퍼센트 목표 후보는 더 높게:

```text
trade count >= 300
profit factor >= 1.35
expectancy 안정
drawdown 회복력 확인
```

# 최종 권장 세팅

처음은 이렇게 시작하는 게 좋다.

```text
전략: 4개
Pullback / Breakout / VWAP / Futures Short

봉: 15분 중심
보조: 5분, 1시간

종목: 거래대금 상위 + 변동성 + 스프레드 필터

탐색: 넓게
승격: 엄격하게

진입: strong 이상
청산: ATR stop + partial TP + trailing + time stop

risk: 작게 시작
capital: 성과 확인 후 자동 집중
```

핵심은 이거다.

```text
초기 세팅은 돈을 벌기 위한 세팅이 아니라,
edge가 있는 조합을 빠르게 찾아내기 위한 세팅이어야 한다.
```
